🎹 Musicology Behind the Generator

This groove box models avant-garde jazz theory using Web Audio synthesis. First, it sets up a consonant jazz framework using chords rich in tritone intervals—such as the Lydian Maj7#11 (housing a beautiful tritone between the 1st and #11th degrees, e.g. C to F#) or a Dominant 7#11 (housing a dual tritone structure, generating high functional drive).

To maintain arpeggio movement and structural contrast, the algorithm inserts a Tense Bridge at user-defined bar frequencies (e.g., every 4 bars). These bridge chords/scales leverage highly expressive Japanese traditional scales (such as Hirajoshi: a beautiful five-note hemitonic scale containing dark semitones, e.g. C, D, Eb, G, Ab) or Altered Pentatonics. These bridge pentatonics serve as an intense, suspenseful bridge before resolving cleanly back to the lush consonant base.
